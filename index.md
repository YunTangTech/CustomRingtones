## Welcome to Custom Ringtones Support Page

Create your own ringtone for phone. Without connection to the computer, the ringtone can be produced via iPhone.
You can get any ringtone you want by this app.

First, import the recorded video. 
Second, extract audio from the video and make ringtones.
Third, install ringtones through the system application GarageBand.
